# Description #

It is the project that takes pictures of users on Github.

### Setup ###

* git clone https://mar3q@bitbucket.org/mar3q/imgtaker.git
* cd imgtaker
* npm init
* npm install --save express ejs
* node app.js
* Enter the localhost:3000
